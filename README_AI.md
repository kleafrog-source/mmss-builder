# AI Ассистент MMSS - Инструкция по настройке

## Настройка Mistral API

### 1. Получение API ключа

1. Зарегистрируйтесь на https://console.mistral.ai/
2. Создайте API ключ в личном кабинете
3. Скопируйте ключ

### 2. Создание файла .env

Создайте файл `.env` в корне проекта со следующим содержимым:

```env
MISTRAL_API_KEY=your_mistral_api_key_here
SECRET_KEY=your_secret_key_here
```

**Важно:** 
- Замените `your_mistral_api_key_here` на ваш реальный API ключ
- Не коммитьте файл `.env` в git (он уже должен быть в .gitignore)

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Перезапуск сервера

После добавления `.env` файла перезапустите Flask сервер:

```bash
python run.py
```

## Использование AI Ассистента

### Доступ к разделу:

1. Откройте http://localhost:5000
2. Перейдите в "AI Ассистент" в меню навигации
3. Или прямой URL: http://localhost:5000/ai

### Функциональность:

1. **Автоматическая генерация промпта:**
   - Введите ваш запрос
   - Система автоматически создаст MMSS промпт
   - Промпт будет отправлен в Mistral AI

2. **Настройка промпта:**
   - Выберите тип промпта (полный, PFR, FRP, A-MMSS)
   - Выберите домен применения (70 доступных)
   - Промпт будет адаптирован под выбранные параметры

3. **Отправка в Mistral:**
   - Нажмите "Отправить в Mistral AI"
   - AI будет работать в контексте MMSS системы
   - Ответ будет отформатирован и отображен

4. **Только генерация промпта:**
   - Можно сгенерировать промпт без отправки в AI
   - Полезно для копирования и использования в других системах

## Troubleshooting

### Проблема: "Mistral API не инициализирован"

**Решение:**
1. Проверьте, что файл `.env` существует в корне проекта
2. Убедитесь, что `MISTRAL_API_KEY` указан правильно
3. Перезапустите сервер после создания/изменения `.env`

### Проблема: "Import mistralai could not be resolved"

**Решение:**
```bash
pip install mistralai
```

### Проблема: Ошибка API при отправке

**Решение:**
1. Проверьте, что API ключ валиден
2. Убедитесь, что у вас есть доступ к Mistral API
3. Проверьте баланс на аккаунте Mistral

## API Endpoints

- `GET /ai` - Страница AI ассистента
- `POST /api/ai/generate-and-send` - Генерация промпта и отправка в Mistral
- `POST /api/ai/generate-prompt-only` - Только генерация промпта
