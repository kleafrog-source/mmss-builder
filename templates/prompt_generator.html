Информация о генераторе

Генератор промптов создает структурированные промпты для нейросетей (LLM), основанные на принципах MMSS системы:

  • Полный промпт - включает все компоненты MMSS и метрики
  • PFR промпт - специализированный для фрактальной пересборки
  • FRP промпт - для темпоральной навигации по рекурсивным сценариям
  • A-MMSS промпт - для плетения контекста с этической стабилизацией

Сгенерированные промпты можно использовать с ChatGPT, Claude, Gemini и другими LLM.
