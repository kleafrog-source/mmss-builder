# 🚀 Быстрый запуск MMSS System

## Шаг 1: Установка зависимостей (уже выполнено ✅)

```bash
pip install -r requirements.txt
```

## Шаг 2: Запуск веб-сервера

### Простой способ:
```bash
python run.py
```

### Или альтернативный:
```bash
python app.py
```

## Шаг 3: Открыть в браузере

После запуска откройте:
- **http://localhost:5000**
- или **http://127.0.0.1:5000**

## Что дальше?

1. **Активируйте систему** - нажмите "Активировать MMSS" на главной странице
2. **Выберите модуль:**
   - **PFR** - Фрактальная пересборка
   - **FRP** - Темпоральная навигация  
   - **A-MMSS** - Плетение контекста
3. **Заполните параметры** и запустите расчет

## Остановка сервера

Нажмите `Ctrl+C` в терминале
