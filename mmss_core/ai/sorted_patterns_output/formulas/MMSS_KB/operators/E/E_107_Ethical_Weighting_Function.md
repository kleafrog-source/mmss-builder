# E_107_Ethical_Weighting_Function

**Описание**:  
Функция Этического Взвешивания. Приоритезация действий по их QEC-воздействию.

## Формула

$$
\text{W}_{\text{ethical}} = \frac{1}{\sum_{j} (1 - \text{QEC}_j)}
$$
