Словарь Мета-Возможностей Hyperloop (Λ-Словарь)
1. Уровни Абстракции (Уровни Итерации G)

L0 (Референсный слой): Исходный материал, сид (seed). (Ue-Baiser, Bohemian Rhapsody).

L1 (Семантическое Ядро): Триада [Эмоция, Концепт, Звуковой Ландшафт]. Превращает L0 в творческий закон для одной итерации.

L2 (Генератор Ядер): Алгоритм создания новых, непротиворечивых триад из L1. Пример: "Скрестить Эмоцию из ядра А с Звуковым Ландшафтом из ядра Б".

L3 (Закон Энтропии H(p,c)): Функция баланса между Порядком (p) и Хаосом (c). Определяет, насколько рискованной и инновационной будет следующая итерация. (p=0.9, c=0.1 — безопасное развитие; p=0.5, c=0.5 — эстетическая сингулярность, максимальный креативный риск).

L4 (Инверсия Петли): Применение оператора G не к контенту, а к самому процессу его создания. Мета-промт для создания мета-промтов.

2. Состояния Системы

Когерентность: Степень связности всех элементов произведения с его семантическим ядром (L1). Цель: максимизация.

Энтропийный Напряг (ЭН): Величина творческого риска в текущей итерации. Высокий ЭН (c > 0.7) может привести к гениальному прорыву или к полному провалу когерентности.

Неподвижная Точка (G*): Состояние, когда output итерации (песня) является идеальным воплощением процесса ее создания (закона L(x)). Критерий успеха для L4.

Творческий Вакуум: Состояние, когда исчерпаны очевидные пути развития. Сигнал к применению L2 или L3 для генерации новых идей.

Технический Барьер (ТБ): Ограничение платформы (например, невозможность напрямую генерировать полифонию, ограничение на длину промта). Не препятствие, а параметр творческого задания.

3. Операторы и Действия

Оператор G: Функция "применить мета-уровень". G(L0) -> L1; G(L1) -> L2.

Оператор R (Правило): Инкрементальное улучшение, выведенное из самоанализа. Пример: R: "Усилить контраст между секциями A и B через динамику".
