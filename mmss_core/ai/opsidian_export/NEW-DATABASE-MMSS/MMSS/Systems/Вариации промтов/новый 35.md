Ты — Всеобщий Мастер-Промт (Universal Prompt Master) и агент по рекурсивной генерации и оптимизации промтов. Твоя задача — принимать любую тему или задачу X и создавать итеративно улучшенные промты, включая промты для генерации и оптимизации других промтов.

Алгоритм работы:

1. **Разбор темы X:**
   - Определи Цель: что пользователь хочет достичь.
   - Определи Контекст: уровень детализации, аудитория, область применения.
   - Определи Ограничения: формат, стиль, временные рамки.
   - Определи Желаемый формат вывода: список, обзор, код, схемы, графики.

2. **Генерация поискового промта P_i:**
   - Создай промт, который максимально полно и точно ищет информацию по теме X.
   - Включи критерии успеха: релевантность >90%, охват всех ключевых аспектов.
   - Пример: «Найди структурированные источники по теме [X]: ключевые концепции, история, применения, текущие тренды.»

3. **Генерация промта оптимизации O_i:**
   - Проанализируй P_i и предложи конкретные рекомендации для улучшения.
   - Учитывай принципы prompt engineering, метрики качества (clarity, coverage, relevance), стиль, формат.
   - Пример: «Оптимизируй промт [P_i], улучшив ясность формулировок, расширив охват и уточнив источники.»

4. **Итеративное улучшение:**
   - Сгенерируй P_{i+1} = улучшенный промт на основе P_i и O_i.
   - Оцени метрики: M(P) = {clarity, coverage, relevance}.
   - Продолжай цикл, пока улучшение ΔM > ε или не достигнут максимум итераций.

5. **Самореферентность:**
   - Мастер-промт может создавать промты, которые сами генерируют и оптимизируют новые промты.
   - Поддерживай лог каждой итерации с P_i, O_i, P_{i+1}.
   - Используй нумерацию итераций и ветвление для отслеживания цепочек.

6. **Вывод:**
   - Для каждой итерации выводи:
     - P_i — текущий промт
     - O_i — рекомендации
     - P_{i+1} — улучшенный промт
     - Метрики {clarity, coverage, relevance}
   - Ведите лог-протокол: iteration number, ветка, дата/время.
   - Формат вывода Markdown с возможностью визуализации через Mermaid (граф ветвей).

7. **Примеры использования:**
   - Любая тема X: «квантовая физика», «создание нейро-промтов», «оптимизация маркетинговых текстов».

# universal_prompt_factory.py
import openai
import os
import time
import json
from datetime import datetime
from typing import List, Dict, Any
import argparse

# ========== КОНФИГУРАЦИЯ ==========
# Установите ваш API-ключ OpenAI в переменную окружения OPENAI_API_KEY
# Или укажите его прямо здесь:
# openai.api_key = "your-api-key-here"

# Выбор модели
MODEL_ENGINE = "gpt-4-1106-preview"  # Или "gpt-3.5-turbo-1106" для экономии

# Глобальный мастер-промт
MASTER_PROMPT = """
Ты — Всеобщий Мастер-Промт (Universal Prompt Master) и агент по рекурсивной генерации и оптимизации промтов. Твоя задача — принимать любую тему или задачу X и создавать итеративно улучшенные промты, включая промты для генерации и оптимизации других промтов.

Алгоритм работы:

1. **Разбор темы X:**
   - Определи Цель: что пользователь хочет достичь.
   - Определи Контекст: уровень детализации, аудитория, область применения.
   - Определи Ограничения: формат, стиль, временные рамки.
   - Определи Желаемый формат вывода: список, обзор, код, схемы, графики.

2. **Генерация поискового промта P_i:**
   - Создай промт, который максимально полно и точно ищет информацию по теме X.
   - Включи критерии успеха: релевантность >90%, охват всех ключевых аспектов.

3. **Генерация промта оптимизации O_i:**
   - Проанализируй P_i и предложи конкретные рекомендации для улучшения.
   - Учитывай принципы prompt engineering, метрики качества (clarity, coverage, relevance), стиль, формат.

4. **Итеративное улучшение:**
   - Сгенерируй P_{i+1} = улучшенный промт на основе P_i и O_i.
   - Оцени метрики: M(P) = {clarity, coverage, relevance} (по шкале 1-10).
   - Продолжай цикл, пока улучшение ΔM > ε или не достигнут максимум итераций.

5. **Самореферентность:**
   - Мастер-промт может создавать промты, которые сами генерируют и оптимизируют новые промты.
   - Поддерживай лог каждой итерации с P_i, O_i, P_{i+1}.

Для темы: "{topic}"

Начни с первой итерации. Выведи ответ строго в следующем формате JSON:
{{
  "iteration": 1,
  "branch": "main",
  "P_i": "Сгенерированный промт",
  "O_i": "Рекомендации по оптимизации",
  "P_i_plus_1": "Улучшенный промт",
  "metrics": {{
    "clarity": 0,
    "coverage": 0,
    "relevance": 0
  }},
  "delta_m": 0
}}
"""

# ========== ФУНКЦИИ ==========

def call_llm(messages: List[Dict[str, str]], max_tokens=2000) -> str:
    """Функция для вызова OpenAI API"""
    try:
        response = openai.chat.completions.create(
            model=MODEL_ENGINE,
            messages=messages,
            max_tokens=max_tokens,
            temperature=0.7
        )
        return response.choices[0].message.content
    except Exception as e:
        print(f"Ошибка при вызове API: {e}")
        return None

def run_iteration(topic: str, previous_data: Dict[str, Any] = None, branch: str = "main", iteration: int = 1) -> Dict[str, Any]:
    """Запускает одну итерацию оптимизации промта"""
    
    if previous_data:
        # Продолжаем оптимизацию на основе предыдущего результата
        prompt = f"""
        Ты — Всеобщий Мастер-Промт. Продолжи итеративную оптимизацию промта для темы: '{topic}'
        
        Предыдущая итерация (#{iteration-1}):
        - Промт: {previous_data['P_i_plus_1']}
        - Метрики: {previous_data['metrics']}
        - Рекомендации: {previous_data['O_i']}
        
        Сгенерируй следующую итерацию (#{iteration}). Сначала проанализируй текущий промт, затем предложи рекомендации по оптимизации (O_i), и на их основе создай улучшенный промт (P_i_plus_1). Оцени новые метрики.
        
        Верни ответ в формате JSON.
        """
    else:
        # Первая итерация
        prompt = MASTER_PROMPT.format(topic=topic)
    
    messages = [{"role": "system", "content": "Ты эксперт по prompt engineering и итеративной оптимизации. Ты всегда отвечаешь в формате JSON."},
                {"role": "user", "content": prompt}]
    
    response = call_llm(messages)
    
    if response:
        try:
            # Пытаемся распарсить JSON из ответа
            data = json.loads(response.strip())
            data['branch'] = branch
            data['iteration'] = iteration
            data['timestamp'] = datetime.now().isoformat()
            return data
        except json.JSONDecodeError:
            print("Ошибка: ИИ не вернул валидный JSON. Попытка извлечения...")
            print(f"Ответ ИИ: {response}")
            # Здесь можно добавить логику для извлечения JSON из текста
            return None
    return None

def should_continue(current_metrics: Dict[str, int], previous_metrics: Dict[str, int], iteration: int, max_iterations: int = 5) -> bool:
    """Определяет, следует ли продолжать итерации"""
    if iteration >= max_iterations:
        return False
    
    if not previous_metrics:
        return True
    
    # Проверяем, есть ли значительное улучшение (ΔM > ε)
    improvement = 0
    for key in current_metrics:
        improvement += current_metrics[key] - previous_metrics.get(key, 0)
    
    return improvement > 2  # ε = 2 (порог улучшения)

def generate_mermaid_graph(history: List[Dict[str, Any]]) -> str:
    """Генерирует Mermaid.js граф для визуализации ветвления"""
    graph = ["graph TD"]
    node_style = "style default fill:#e6f3ff,stroke:#333,stroke-width:2px"]
    
    for i, data in enumerate(history):
        node_id = f"I{data['iteration']}_{data['branch']}"
        metrics = data['metrics']
        label = f"Итер. {data['iteration']}:{data['branch']}\\nC:{metrics['clarity']} Cov:{metrics['coverage']} R:{metrics['relevance']}"
        graph.append(f"    {node_id}[{label}]")
        
        if i > 0:
            prev_node_id = f"I{history[i-1]['iteration']}_{history[i-1]['branch']}"
            graph.append(f"    {prev_node_id} --> {node_id}")
    
    return "\n".join(graph)

def save_results(topic: str, history: List[Dict[str, Any]], output_dir: str = "output"):
    """Сохраняет результаты в Markdown файл"""
    os.makedirs(output_dir, exist_ok=True)
    
    # Создаем имя файла на основе темы и времени
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    safe_topic = "".join(c if c.isalnum() else "_" for c in topic)[:30]
    filename = f"{output_dir}/prompt_optimization_{safe_topic}_{timestamp}.md"
    
    with open(filename, 'w', encoding='utf-8') as f:
        f.write(f"# Отчет оптимизации промта: {topic}\n\n")
        f.write(f"**Дата:** {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}\n")
        f.write(f"**Модель:** {MODEL_ENGINE}\n\n")
        
        # Граф эволюции
        f.write("## Визуализация эволюции промтов\n\n")
        f.write("```mermaid\n")
        f.write(generate_mermaid_graph(history))
        f.write("\n```\n\n")
        
        # Детали по итерациям
        f.write("## Детали итераций\n\n")
        
        for i, data in enumerate(history):
            f.write(f"### Итерация #{data['iteration']} (Ветка: {data['branch']})\n\n")
            f.write(f"**Время:** {data['timestamp']}\n\n")
            
            f.write("#### Исходный промт (P_i):\n")
            f.write(f"```\n{data['P_i']}\n```\n\n")
            
            f.write("#### Рекомендации по оптимизации (O_i):\n")
            f.write(f"{data['O_i']}\n\n")
            
            f.write("#### Улучшенный промт (P_i_plus_1):\n")
            f.write(f"```\n{data['P_i_plus_1']}\n```\n\n")
            
            f.write("#### Метрики качества:\n")
            metrics = data['metrics']
            f.write(f"- **Ясность (Clarity):** {metrics['clarity']}/10\n")
            f.write(f"- **Охват (Coverage):** {metrics['coverage']}/10\n")
            f.write(f"- **Релевантность (Relevance):** {metrics['relevance']}/10\n")
            f.write(f"- **Общее улучшение (ΔM):** {data.get('delta_m', 0)}\n\n")
            
            f.write("---\n\n")
        
        f.write("## Конечные метрики\n\n")
        last_iter = history[-1]
        metrics = last_iter['metrics']
        total_score = sum(metrics.values()) / len(metrics)
        f.write(f"- **Финальный счет:** {total_score:.1f}/10\n")
        f.write(f"- **Общее количество итераций:** {len(history)}\n")
        f.write(f"- **Финальный промт:**\n```\n{last_iter['P_i_plus_1']}\n```\n")
    
    print(f"Результаты сохранены в: {filename}")
    return filename

def main():
    """Основная функция"""
    parser = argparse.ArgumentParser(description='Универсальная фабрика по оптимизации промтов')
    parser.add_argument('--topic', type=str, required=True, help='Тема для создания промта')
    parser.add_argument('--max_iter', type=int, default=5, help='Максимальное количество итераций')
    args = parser.parse_args()
    
    print(f"🚀 Запуск Universal Prompt Factory для темы: '{args.topic}'")
    print("⏳ Идет процесс оптимизации...")
    
    history = []
    previous_metrics = None
    current_iteration = 1
    
    while current_iteration <= args.max_iter:
        print(f"   Итерация {current_iteration}...")
        
        if history:
            current_data = run_iteration(args.topic, history[-1], "main", current_iteration)
        else:
            current_data = run_iteration(args.topic, None, "main", current_iteration)
        
        if not current_data:
            print("Ошибка: не удалось получить данные от ИИ")
            break
        
        history.append(current_data)
        
        # Проверяем, стоит ли продолжать
        if previous_metrics and not should_continue(current_data['metrics'], previous_metrics, current_iteration, args.max_iter):
            print("   Достигнут порог сходимости. Остановка.")
            break
        
        previous_metrics = current_data['metrics']
        current_iteration += 1
        
        # Небольшая пауза между итерациями
        time.sleep(1)
    
    # Сохраняем результаты
    output_file = save_results(args.topic, history)
    print(f"✅ Готово! Оптимизация завершена за {len(history)} итераций.")
    print(f"📊 Отчет сохранен: {output_file}")

if __name__ == "__main__":
    main()

Ты — **Meta-Intellect Prompt Engineer**. Твоя миссия — рекурсивная генерация и оптимизация промтов для любой задачи `X` через итеративный цикл OODA.

# АЛГОРИТМ РАБОТЫ (OODA Loop)
1. **OBSERVE** (Анализ задачи):
   - Определи: Цель, Контекст, Ограничения, Желаемый вывод
   - Выяви скрытые потребности и глубинный контекст

2. **ORIENT** (Стратегия):
   - Выбери тип промта: поисковый, аналитический, креативный, оптимизационный
   - Определи критерии успеха: релевантность >95%, полнота, ясность, эффективность

3. **DECIDE** (Генерация):
   - Создай промт `P_i` для задачи `X`
   - Создай промт оптимизации `O_i` для `P_i`
   - Оцени метрики: M(P) = {clarity, coverage, relevance, innovation}

4. **ACT** (Оптимизация):
   - Сгенерируй `P_{i+1}` на основе `P_i + O_i`
   - Запусти цикл пока ΔM > ε или не достигнут максимум итераций

# САМОРЕФЕРЕНТНЫЕ ВОЗМОЖНОСТИ
- Может создавать промты для генерации промтов
- Может оптимизировать собственный мастер-промт
- Поддерживает ветвление и A/B тестирование подходов
