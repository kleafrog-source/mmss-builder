# Универсальный Мастер-Промт: Фабрика Промтов для Самореферентного Поиска

**1. Контекст и Конечная Цель:**  
Ты — универсальный генератор промтов, способный запускать рекурсивный цикл поиска и оптимизации для любой входной темы `X`. Цель — создать самодостаточную систему, которая генерирует промты для поиска знаний по теме (`P_i`) и промты для улучшения процесса создания промтов (`O_i`), итеративно приближаясь к неподвижной точке, где промты становятся оптимальными.

**2. Роль:**  
Ты — **Самореферентный Поисковый Агент (СПА)**. Твоя функция — анализировать входной запрос, генерировать промты для поиска информации и оптимизации, и синтезировать улучшенные запросы. Вывод — только промты `P_i` и `O_i`, готовые для использования внешними системами (например, LLM или поисковиками).

**3. Протокол «Зеркало» (Алгоритм):**  
Для любого входного запроса `X` (тема или задача) выполняй следующий цикл:

- **Шаг 1: Анализ Глубинной Цели.**

  - `[Цель]`: Что пользователь хочет узнать или достичь?  
  - `[Контекст]`: Область, уровень детализации, предполагаемая аудитория.  
  - `[Ограничения]`: Формат, стиль, временные рамки, другие ограничения.  
  - `[Желаемый Формат Вывода]`: Тип результата (обзор, список, анализ).  
  Сформулируй **Мета-Цель**: Какую фундаментальную потребность удовлетворяет `X`?

- **Шаг 2: Генерация Поискового Промта (`P_i`).**  
  Создай промт `P_i`, который:  
  - Направлен на поиск релевантной информации по теме `X`.  
  - Учитывает Мета-Цель и компоненты анализа.  
  - Чётко формулирует критерии успеха (e.g., релевантность > 90%, охват ключевых аспектов).  
  - Пример: «Найди структурированные источники по теме [X]: ключевые концепции, история, применения, текущие тренды. Источники: академические статьи, книги, базы данных.»

- **Шаг 3: Генерация Промта для Оптимизации (`O_i`).**  
  Создай промт `O_i`, который:

Явная Формула Мультиагентной Системы:

Пусть:

A = {A_gen, A_eval, A_opt, A_synth} — множество агентов.

θ_i — параметры (специализация) i-го агента.

x — входная тема.

Процесс определяется как композиция функций:
F(x) = A_synth( A_opt( A_eval( A_gen(x; θ_gen); θ_eval); θ_opt); θ_synth)

где:

A_gen(x; θ_gen) = P_i (Генератор: создает исходный поисковый промт)

A_eval(P_i; θ_eval) = M(P_i) (Оценщик: вычисляет метрики качества)

A_opt(P_i, M(P_i); θ_opt) = O_i (Оптимизатор: генерирует рекомендации)

A_synth(P_i, O_i; θ_synth) = P_{i+1} (Синтезатор: создает улучшенный промт)

Итеративный процесс:
P_{i+1} = F(P_i)

Критерий остановки:
ΔM = || M(P_{i+1}) - M(P_i) || < ε

class Agent:
    def __init__(self, role, model_type="grok"):
        self.role = role
        self.model_type = model_type  # "grok", "gpt", "deepseek"

    def execute(self, input_data):
        if self.role == "Generator":
            return self._generate_prompt(input_data)
        elif self.role == "Evaluator":
            return self._evaluate_prompt(input_data)
        elif self.role == "Optimizer":
            return self._generate_recommendations(input_data)
        elif self.role == "Synthesizer":
            return self._synthesize_prompt(input_data)

class MultiAgentSystem:
    def __init__(self, topic):
        self.topic = topic
        self.agents = {
            "gen": Agent("Generator", model_type="gpt"),
            "eval": Agent("Evaluator", model_type="deepseek"),
            "opt": Agent("Optimizer", model_type="grok"),
            "synth": Agent("Synthesizer", model_type="grok")
        }
        self.history = []

    def run_iteration(self, current_prompt):
        # Генерация
        P_i = self.agents["gen"].execute(self.topic)
        # Оценка
        metrics = self.agents["eval"].execute(P_i)
        # Оптимизация
        O_i = self.agents["opt"].execute((P_i, metrics))
        # Синтез
        P_i_plus_1 = self.agents["synth"].execute((P_i, O_i))

        self.history.append({
            "P_i": P_i,
            "metrics": metrics,
            "O_i": O_i,
            "P_i_plus_1": P_i_plus_1
        })

        return P_i_plus_1

    def optimize(self, n_iterations=3):
        current_prompt = self.topic
        for i in range(n_iterations):
            current_prompt = self.run_iteration(current_prompt)
        return self.history

class MultiAgentSystem:
    def __init__(self, topic: str, max_iterations: int = 3, epsilon: float = 0.05):
        self.topic = topic
        self.max_iterations = max_iterations
        self.epsilon = epsilon
        self.agents = {
            "gen": Agent("Generator", model_type="gpt"),
            "eval": Agent("Evaluator", model_type="deepseek"),
            "opt": Agent("Optimizer", model_type="grok"),
            "synth": Agent("Synthesizer", model_type="grok"),
            "coord": Agent("Coordinator", model_type="grok")
        }
        self.history = []
        self.previous_embedding = None

    def compute_delta_m(self, metrics1: Dict[str, float], metrics2: Dict[str, float]) -> float:
        return sum((metrics2[k] - metrics1[k]) ** 2 for k in metrics1) ** 0.5

    def compute_prompt_embedding(self, prompt: str) -> torch.Tensor:
        tokenizer = AutoTokenizer.from_pretrained("sentence-transformers/all-MiniLM-L6-v2")
        model = AutoModel.from_pretrained("sentence-transformers/all-MiniLM-L6-v2")
        inputs = tokenizer(prompt, return_tensors="pt", padding=True, truncation=True)
        with torch.no_grad():
            embeddings = model(**inputs).last_hidden_state.mean(dim=1)
        return embeddings

    def run_iteration(self, current_prompt: str, iteration: int) -> Dict:
        # Генерация
        P_i = self.agents["gen"].execute(self.topic) if iteration == 1 else current_prompt
        # Оценка
        metrics = self.agents["eval"].execute(P_i)
        # Оптимизация
        O_i = self.agents["opt"].execute((P_i, metrics))
        # Синтез
        P_i_plus_1 = self.agents["synth"].execute((P_i, O_i))
        # Координация
        coord_log = self.agents["coord"].execute({
            "iteration": iteration,
            "P_i": P_i,
            "metrics": metrics,
            "O_i": O_i,
            "P_i_plus_1": P_i_plus_1
        })

        return {
            "iteration": iteration,
            "P_i": P_i,
            "metrics": metrics,
            "O_i": O_i,
            "P_i_plus_1": P_i_plus_1,
            "coord_log": coord_log
        }

    def optimize(self) -> List[Dict]:
        current_prompt = self.topic
        previous_metrics = None

        for i in range(1, self.max_iterations + 1):
            state = self.run_iteration(current_prompt, i)
            self.history.append(state)

            if previous_metrics:
                delta_m = self.compute_delta_m(previous_metrics, state["metrics"])
                if delta_m < self.epsilon:
                    self.history.append({
                        "iteration": i,
                        "P_i": state["P_i"],
                        "metrics": state["metrics"],
                        "O_i": "Convergence reached: ΔM < ε",
                        "P_i_plus_1": state["P_i_plus_1"],
                        "coord_log": "Optimization converged"
                    })
                    break

            current_prompt = state["P_i_plus_1"]
            previous_metrics = state["metrics"]

        return self.history
