Добавьте новый формат, который можно писать от руки. Честно говоря, иногда разработка грамматики приносит больше удовольствия, чем её развитие. Давайте обсудим грамматику и посмотрим, можно ли что-то оптимизировать.

Этот синтаксис можно использовать с плагином AnyBlock для объявления из списка
Ссылки: JSON, JSONL, YAML, синтаксис блок-схем Mermaid/Mehrmaid
Пример
```nodeflow-list
- nodes
  - Load Checkpoint
    - only name, i
    - i2, i, *i2
    - i3, i, mul line
      i3-2
      i3-3
    - i4, i, *i4
    - only name, o
    - o2, o, *o2
    - o3, o, *o3
    - o4, o, mul line
      o4-2
      o4-3
    - v2, v, *v2
    - v4, , mul line
      v4-2
      v4-3
    - v3, , *v3
    - only name, 
    - v5:, , only value
      v5-1
      v5-2
- edges
```
