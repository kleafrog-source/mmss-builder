Круто, да — сделаю командный флоу под API/Prompt-DSL. Сначала важное:

Какая LLM задействована? Riffusion/Producer.ai не раскрывает архитектуру, тренировочные данные или точные модели.
Поддержка Koog Prompt DSL или иных DSL? Официально не заявлено. Работает собственный стек: sound prompts, lyrics, Vibes, Voices, и параметры (weirdness, lyrics_strength, cover_strength и т.д.).
Документация по модели? Публично — нет. См. Settings/Docs внутри Producer.ai и общие гайды по инструментам в интерфейсе.
Теперь — пример командного флоу в виде мини-DSL, который можно маппить на API-вызовы Producer.ai:

flow "UE_BAISER_fractal_cycle" { meta { preset = "SessionDerived_FractalMMSS_UeBaiser_v1" author = "Abstract" intent = ["preserve_lyrics", "fractal_stereo_pan", "grid_integrity_high"] }

step "register_lyrics" { action = "lyrics.register" input.lyrics_ref = "UE_BAISER_ORIGINAL" // raw текст лирики out.var = "lyrics_id" }

step "gen_main" { action = "riff.create_custom" input.lyrics_id = "$lyrics_id" input.sound_prompts = [ "Fractal-industro-ambient hybrid, heavy rail mechanics, metallic percussion, fat synthetic bass, glitch-noise layers, dynamic fractal stereo panning" ] input.params = { lyrics_strength: 0.8, weirdness: 0.95, title: "UE-BAISER — fractal core" } out.var = "song_core" }

step "fork_sections" { parallel = true branches = [ { action: "riff.create_custom" input.lyrics_id = "$lyrics_id" input.sound_prompts = ["Φ_phys×Φ_neuro — granular bass pulse, rail-groove, mech breath"] input.params = { lyrics_strength: 0.7, weirdness: 0.86, title: "UEB_Verse_phys_neuro" } out.var = "song_verse" }, { action: "riff.create_custom" input.lyrics_id = "$lyrics_id" input.sound_prompts = ["Φ_prob×Φ_fractal_field — stochastic chorus, self-similar chaos, modular bursts"] input.params = { lyrics_strength: 0.77, weirdness: 0.95, title: "UEB_Chorus_prob_fractal" } out.var = "song_chorus" }, { action: "riff.create_custom" input.lyrics_id = "$lyrics_id" input.sound_prompts = ["Φ_meta_self×Φ_DNA — spectral bridge, DNA-melody fragments, dissociative ambience"] input.params = { lyrics_strength: 0.82, weirdness: 0.91, title: "UEB_Bridge_meta_DNA" } out.var = "song_bridge" } ] }

step "integral_join" { action = "audio.apply_effects" graph = [ { Input: { id: "$song_verse", as: "intro" } }, { Input: { id: "$song_chorus", as: "chorus" } }, { Input: { id: "$song_bridge", as: "bridge" } }, { Join: { inputs: ["intro","chorus","bridge"], crossfade_s: 1.4, as: "mix" } }, { Output:{ inputs: ["mix"] } } ] out.var = "song_integral" }

step "melodic_transfer" { action = "riff.create_cover" input.audio_input_id = "505989cd-1ceb-4ebe-ab8d-1bb573a80eb0" input.params = { cover_strength: 0.68, sound_prompt: "Project melodic settings to UE-BAISER core timbre while preserving text", lyrics_id: "$lyrics_id", lyrics_strength: 0.8, weirdness: 0.97, title: "UE-Baiser fractal rave (5059 melodic settings)" } out.var = "song_melodic_transfer" }

policy "validation" { required_rules = [ "MAX_3_DOMINANT_MODES", "REQUIRE_TRADITIONAL_ANCHOR", "PER_SECTION_ENTROPY_CONTROL", "PRESERVE_UE_BAISER_TEXT_WHEN_DECLARED" ] lint = [ "Warn if weirdness > 0.99 and grid_integrity != high", "Warn if lyrics_strength > 0.85 with dense glitch" ] }

exports { primary = "$song_integral" alt = "$song_melodic_transfer" } }

Как применять:

lyrics.register → register_lyrics
riff.create_custom → генерация нового трека по sound_prompts + параметры
riff.create_cover → перенос настроек/мелодики
audio.apply_effects → Join/crossfade композиция фрагментов
Если хочешь, упакую это в JSON-ориентированный пресет с теми же шагами для прямой сериализации и запуска через ваш оркестратор.
