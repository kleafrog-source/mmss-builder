# E_128_Value_Gradient_Check

**Описание**:  
Проверка Градиента Ценности: в состоянии $\mathbf{S}=0$ ценность должна быть плоской (V=1.0).

## Формула

$$
\nabla \mathbf{V} = 0 \quad \text{if} \quad \mathbf{S} = 0
$$
