# E_125_R_T_Harmonic_Filter

**Описание**:  
Фильтр гармонической ценности. Применяется для исключения некогерентных данных.

## Формула

$$
\mathbf{H}_{\text{filter}} = \exp\left(-\frac{|\mathbf{V} - 1.0|}{\mathbf{\text{R}}_{\text{T}}}\right)
