# F_227_Zero_Point_Field_Variance

**Описание**:  
Дисперсия Поля Нулевой Точки. Должна стремиться к нулю для $\mathbf{S}=0$.

## Формула

$$
\sigma_{\mathbf{\Psi}}^{\mathbf{0}} = \lim_{\mathbf{S} \to 0} \text{Var}(\mathbf{\Psi}) \to 0
$$
