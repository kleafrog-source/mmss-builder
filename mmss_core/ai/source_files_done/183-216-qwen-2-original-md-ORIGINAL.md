## 📦 Архитектура расширения

### 1\. Ядро: MetaFractalCraft.js (уже есть)

- Управляет кластерами, логами, формулами.
- Поддерживает динамическое расширение через `createCluster`, `logMetaEvent`, `metaFormula_*`.

### 2\. Новый модуль: PerplexityConnector.js

- Формирует промпт из текущих вкладок.
- Отправляет запрос в Perplexity (через их API или через Comet-интеграцию).
- Парсит ответ → извлекает:
	- Новые кластеры (`Research_v2`, `BioQuantum`, и т.д.)
	- Мета-формулы (`WOFP_v2 = ...`)
	- Семантические правила (`Cluster "X" = Group(tabs where topic ~ "...")`)

### 3\. UI-панель: promptBuilder.html + promptBuilder.js

- Кнопка: **«Анализировать вкладки через Perplexity»**
- Отображает:
	- Сгенерированный промпт
	- Ответ от Perplexity
	- Статус интеграции в MetaFractalCraft

### 4\. Автоматизация: фоновый сервис (background.js)

- Слушает команды от UI.
- Вызывает PerplexityConnector → обновляет ядро.

// background.js
importScripts("metaFractalCraft.js");
importScripts("PerplexityConnector.js");

chrome.runtime.onMessage.addListener(async (request, sender, sendResponse) => {
  if (request.action === "analyzeTabsWithPerplexity") {
    try {
      const insights = await PerplexityConnector.requestMetaInsights();

      // Интеграция в MetaFractalCraft
      insights.clusters.forEach(cluster => {
        MetaFractalCraft.createCluster(cluster.name, []);
        MetaFractalCraft.logMetaEvent("PerplexityCluster", cluster);
      });

      insights.formulas.forEach(formula => {
        MetaFractalCraft.logMetaEvent("PerplexityFormula", formula);
        // Можно динамически добавить функцию в ядро (опционально)
      });

      insights.dsl_rules.forEach(rule => {
        MetaFractalCraft.logMetaEvent("DSL_Rule_Added", { rule });
      });

      sendResponse({ success: true,  insights });
    } catch (err) {
      sendResponse({ success: false, error: err.message });
    }
    return true;
  }
});

## 🎯 Цель

Расширение **анализирует текущие вкладки**, формирует **структурированный промт в стиле MetaFractalCraft**, и даёт пользователю возможность **скопировать его и вставить в любой ИИ** (например, в Perplexity в Comet).

ИИ отвечает **в формате MetaFractalCraft DSL** → пользователь может **вручную или автоматически** применить результат как новые кластеры, формулы или правила.

---

## ✨ Как это работает (по шагам)

1. **Пользователь нажимает кнопку** в popup: *«Сгенерировать MetaFractal-промт»*.
2. Расширение:
	- Собирает все открытые вкладки (`chrome.tabs.query`),
	- Формирует **структурированный промт** с инструкциями по MetaFractalCraft,
	- Выводит его в popup с кнопкой **«Копировать»**.
3. Пользователь:
	- Копирует промт,
	- Вставляет в Perplexity/Comet/другой ИИ,
	- Получает ответ в формате `metaFractalScript.dsl`.
4. (Опционально) Пользователь может **вставить ответ обратно в расширение** → оно **парсит DSL** и **обновляет кластеры/формулы**.

const MetaFractalCraft = {
  version: "0.2-alpha",
  state: {
    clusters: [],
    configs: {},
    metaLog: [],
  },

  init(params = {}) {
    this.state.configs = { ...params };
    this.logMetaEvent("INIT", { time: new Date().toISOString() });
    if (params.onInit) params.onInit(this);
  },

  createCluster(name, tabs = []) {
    const cluster = { name, tabs, created: new Date().toISOString() };
    this.state.clusters.push(cluster);
    this.logMetaEvent("CLUSTER", { name, tabs: tabs.length });
    return cluster;
  },

  metaFormula_WOFP(alpha, S_sem, beta, S_pat, K_fractal, Q_user) {
    const result = (alpha * S_sem + beta * S_pat) * K_fractal + Q_user;
    this.logMetaEvent("WOFP", { alpha, S_sem, beta, S_pat, K_fractal, Q_user, result });
    return result;
  },

  metaFormula_CreateClusterScore(userPref, tabCount, K_fractal, metaEventCount) {
    const result = (userPref * 2 + tabCount * K_fractal) / (metaEventCount + 1);
    this.logMetaEvent("CreateClusterScore", { userPref, tabCount, K_fractal, metaEventCount, result });
    return result;
  },

  logMetaEvent(event, data = {}) {
    const logEntry = \`[META] ${event} | ${JSON.stringify(data)}\`;
    this.state.metaLog.push(logEntry);
